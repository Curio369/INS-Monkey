# India INS Food Additives — Complete Safety Reference

A single-page, self-contained reference of food additives permitted under India's
**INS (International Numbering System)** codes, with clear safety classifications
(safe / caution / unsafe), search, and filtering — all in one static HTML file.

## 🔗 Live

**Deployed and live.** Open the hosted page or run `India_INS_Food_Additives_Complete-1.html` locally in any modern browser.

## Features

- Complete INS food-additive listing with safety indicators
- Colour-coded legend: 🟢 Safe · 🟡 Caution · 🔴 Unsafe
- Instant client-side search and filtering
- Fully self-contained — no build step, no dependencies, no backend

## Usage

Just open the HTML file in a browser:

```
India_INS_Food_Additives_Complete-1.html
```

That's it — everything (styles, data, logic) is bundled in the single file.
